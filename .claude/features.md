# Features

## Project Intent

DFScrimPlanner helps Delta Force teams coordinate 3-hour practice scrims. Team leaders input their team's availability windows; the app finds overlapping slots and matches teams together. Scrim organizers can search by time (then pick teams) or by teams (then find a common time). Solo players can register as fillers for teams that need a substitute.

## Feature List

| Feature | Description | Status |
|---------|-------------|--------|
| Invite-based registration | Users join via email invite link; accounts activate on first sign-in | planned |
| Team management | Leaders create teams, invite teammates by email, manage roster | planned |
| Teammate account activation | Teammates receive invite; activate account to manage their own availability | planned |
| Availability input | Users set availability windows; stored in UTC, displayed in user's local timezone | planned |
| Filler registration | Solo players can register without joining a team, visible to leaders as available fillers | planned |
| Filler search | Leaders can search for fillers available on a given date/time range | planned |
| Time-first scrim planner | Organizer picks a time slot → app shows which teams are available | planned |
| Teams-first scrim planner | Organizer picks two teams → app shows their common available slots | planned |
| Availability grid UI | Clean shadcn-based visual grid showing team/player availability at a glance | planned |
| Timezone support | Each user stores their timezone; all comparisons done in UTC, display in local time | planned |

## Out of Scope

- In-app chat or messaging
- Automated scrim scheduling (organizer always confirms)
- Payment or premium tiers (at launch)
- Mobile app
